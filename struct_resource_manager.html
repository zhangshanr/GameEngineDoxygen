Detailed Description

Singleton class that manages resources like textures.

The ResourceManager class provides a centralized way to load and manage textures to ensure that each texture is loaded only once. It uses a singleton pattern to provide a global point of access to the resource management functions.

std::string filepath )
inline

Loads an SDL_Texture from a file, or retrieves it if already loaded.

This function checks if a texture for the given filepath has already been loaded. If it has not, it loads the texture, stores it in the map, and returns it. If it has been loaded, it simply returns the existing texture.
